## Steps

### 1. Clone Repository

```bash
git clone https://github.com/Mnuralim/pingpong-bot.git
```

```
cd pingpong-bot
```

### 2. Instal Dependency

```bash
npm install
```

### 3. configurate .env File

```bash
nano .env
```

```bash
PRIVATE_KEYS=["pk1","pk2"]
BEARER_TOKENS=["token1","token2"]


```

### 4. Run script

```bash
npm run start
```

\*Make sure you have Node.js and git installed on your system.
